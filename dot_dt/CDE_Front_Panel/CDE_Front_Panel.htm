Mozilla & CDE

Mozilla CDE Front Panel Action & Icon

When Netscape 4.79 went end-of-life (EOL), I moved to Mozilla as my browser, but missed being able to invoke it from the CDE front panel. So I wrote my own front panel based on the Netscape globe front panel including the clock and an action to kill Mozilla when it acts up on me. Recently I received an email indicating among other things, the CDE Netscape globe front panel would not be changed to Mozilla. Because of that, I decided to share my Mozilla front panel script. This is what I did to install the Mozilla front panel. Use at your own risk .

  • Save mozilla.fp and mozilla.dt to your ~/.dt/types directory and make sure you have read permission for these files.
  • Save mozicon50.pm and dynamite.pm to your ~/.dt/icons directory and make sure you have read permission for these files.
  • Do a button 3 mouse click on the CDE desktop background and select "Restart Workspace Manager".
  • Some screen dumps of what I did, and where I have my files, is below:

Troubleshooting

I have seen problems where CDE doesn't want to allow you to add a new front panel if you've deleted it once before. If you run into problems, you can start your front panel over with default values by doing the following:
  • Rename the current dynamic front panel directories 

    
$ pwd
/home/nygren/.dt/types
$ ls -lad fp_dyna*
drwx--x--x   2 nygren   staff       3584 Aug 31 10:13 fp_dynamic/
drwx--x--x   2 nygren   staff        512 Aug 31 10:12 fp_dynamic1/
$ mv fp_dynamic fp_dynamic_old
$ mv fp_dynamic1 fp_dynamic1_old

  • Do a button 3 mouse click on the CDE desktop background and select "Restart Workspace Manager".
  • The new mozilla icon should come up along with a lot of old stuff you may have previously deleted. Just re-delete the icons you don't want and do another "Restart Workspace Manager" after everything looks close to the way you want it.

Why I still use CDE

I haven't made the change from the Common Desktop Environment (CDE) to the Gnome GUI yet, because I haven't had any real problems with CDE, have a lot invested in custom CDE actions, think Gnome is slower than CDE, and I haven't figured out how to do the following in Gnome yet:

  • Bring up a shell terminal window at the directory Nautilus is currently displaying.
  • Easily show hidden files in Nautilus. (Seems like this is a multi-step process.)
  • Show my floppy disk drive without invoking CDE.
If someone has some good ways to do the above, I'd like to know.
